# CLAUDE Entry Instructions

Use the vendor-neutral `.ai` framework.

Before coding:
1. Read `AI_OS.md`.
2. Read `.ai/config/coding-rules.md`.
3. Read `docs/specs/project/PROJECT_SPEC.md`.
4. Read relevant module and feature SPEC files.
5. Create a plan.
6. Wait for approval.
7. Implement only approved scope.
8. Run tests.
9. Perform self-review.
10. Update documentation if needed.

Never:
- Commit secrets
- Modify unrelated files
- Change API contracts without approval
- Deploy production without explicit approval
