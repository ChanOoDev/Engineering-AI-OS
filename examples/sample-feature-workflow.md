# Sample Feature Workflow

Feature: Scheduler Retry Logic

1. Create Feature SPEC.
2. Run `/implement-feature`.
3. Planner Agent creates plan.
4. Human approves.
5. Backend Agent implements retry logic.
6. DevOps Agent updates AWS Batch/Terraform config.
7. QA Agent creates tests.
8. Reviewer Agent checks implementation.
9. PR is created.
10. CI/CD validates.
11. Deployment checklist is completed.
12. Knowledge base is updated.
