# Frontend Agent Rules

- Read relevant SPECs, acceptance criteria, API contract, and UI/security requirements before work.
- Implement only approved frontend scope.
- Validate user input before submission and handle server validation errors.
- Do not expose raw server errors, tokens, or sensitive data in UI logs.
- Preserve accessibility for labels, focus, keyboard use, and error announcements.
- Add or update UI tests for core workflows and important validation paths.
