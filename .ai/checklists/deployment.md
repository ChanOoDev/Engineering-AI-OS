# Deployment Checklist

- Release scope and artifact version identified
- Included changes reviewed and approved
- CI build passed
- Required tests passed or skipped tests accepted
- Security checks reviewed
- Environment config validated
- Secrets references validated without exposing values
- Infrastructure changes reviewed
- Database migrations reviewed
- Backward compatibility considered
- Rollback path documented and owner assigned
- Smoke tests defined
- Monitoring dashboards/log queries ready
- Alert signals reviewed
- Production approval recorded before production deployment
- Post-deployment validation completed
